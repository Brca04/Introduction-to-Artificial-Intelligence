# Introduction-to-Artificial-Intelligence
This repository contains laboratory exercises from the course [Introduction to Artificial Intelligence](https://www.fer.unizg.hr/en/course/itai) at FER.
## About
There are many cognitive tasks that are simple for the human, but very difficult for the computer. Artificial intelligence (AI) tries to solve these kind of problems. Traditional approach to AI is based on symbolic knowledge representation and reasoning as symbol manipulation. Biologically inspired connectivistic approach simulates cognitive abilities of the brain. The aim of the course is to introduce different AI approaches and to give an overview of AI methods, including methods for knowledge representation, automatic reasoning, problem solving, learning, and optimization.
## Learning Outcomes
- explain the basic concepts and methods of artificial intelligence
- apply search algorithms for problem solving
- represent knowledge using propositional and first-order logic
- apply reasoning and inference methods
- implement machine learning algorithms
- design and train neural networks
- apply optimization techniques to solve complex problems
- analyze and evaluate the performance of AI methods
